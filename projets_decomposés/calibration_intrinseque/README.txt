L'echequier est de type 9x6, on peut extraire la matrice intrinseque avec la commande suivante :
./calibration  -w 9 -h 6 -s 0.025 -o cam.yml imagelist.xml
où imagelist.xml représente la liste des images obtenue avec le code imagelist_creator et cam.yml le fichier de sortie de notre calcul de notre calibration
